int bordasImg ( char  img[50])

Existe a chance de na multiplicação o valor do RGB ultrapase o 255 ou seja menor que 0, por isso foram criados seis if para assim caso o valor seja menor doque zero ele passe a ser zero, e se for um valor superior a 255 o valor seja colocado para 255. Os valores RGB sao adicionados ao valor de uma outra matriz denominada nImg. Apos a leitura a matriz nImg é impresa dentro do arquivo com a imagem modi- ficada usando outra dupla de FOR.
