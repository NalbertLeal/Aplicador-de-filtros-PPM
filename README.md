Funções do Projeto:

	bordas,

	negativo,
	zoom in (reduzir tamanho),
	zoom out (ampliar imagem),
	rotação anti-horario,
	rotação horario

Compilar o arquivo:

	Para compilar pode ser utilizado o Makefile ou pode pode ser usado o comando:  gcc main.c modificacao.c lerEscreverPPM.c -o projeto
